Web Tourist



For security reasons, the Web Tourist extension at chrome web store is required to share a screen.
  • To share your screen, you must use a non-mobile version of the Chrome browser.
  • The Web Tourist screen sharing can be viewed by mobile devices that are WebRTC-compatible.
  • Not compatible with iPhones® and iPads®.
